# 教程模式设计 — TutorialMode

## 一、设计理念

教程不是教玩家怎么开枪（射击是 AI 自动的），也不是教玩家怎么走路。**教程只教一件事：手势指挥**。

```
教程 = 4 个训练阶段，递进解锁

       ① 手势识别训练
          （学会摆 6 个手势）
                ↓
       ② 指挥基础训练
          （学会选人→移动→警戒→集合）
                ↓
       ③ 破门突入训练
          （学会完整的清房流程）
                ↓
       ④ 综合考核
          （放几个简单的敌人，实战走一遍）
```

每一阶段只引入 1~2 个新概念，练熟了才能进下一阶段。

---

## 二、四阶段详细设计

### 阶段 ①：手势识别训练

**目标**：把 6 个手势都摆对，让识别系统能稳定识别。

**场景**：训练大厅进门左侧区域。

```
┌─────────────────────────────────────────┐
│  手势识别区                              │
│                                         │
│  ┌───┐  ┌───┐  ┌───┐  ┌───┐  ┌───┐  ┌───┐  ┌───┐
│  │ ☝️ │  │ ✌️ │  │ 👇 │  │ ✊ │  │ ✋ │  │ 🤚 │  │ 👊 │
│  │ 1  │  │ 2  │  │ 3  │  │ 4  │  │ 5  │  │ 6  │  │ 7  │
│  │选A │  │选B │  │移动│  │破门│  │警戒│  │禁火│  │集合│
│  └───┘  └───┘  └───┘  └───┘  └───┘  └───┘  └───┘
│                                              │
│           💡 当前：第 1 个手势 — 选中队友 A    │
│   伸出 1 根食指，保持 0.5 秒                   │
│                                         │
│         [  ✅  识别成功  ]                │
│                                         │
└─────────────────────────────────────────┘
```

**流程**：

```
① 屏幕显示第 1 个手势的图示 + 说明文字
② 玩家摆出对应手势
③ 系统识别到 → 灯变绿 → "✅ 识别成功"
④ 自动进入第 2 个手势
⑤ 7 个全部成功 → "🎉 手势训练完成，进入下一阶段"
```

**要点**：
- 每个手势必须持续保持 0.5 秒以上才算"成功"
- 如果某个手势连续 3 次识别失败，提示"请尝试调整手指角度"
- 7 个全部通过才能解锁阶段②

---

### 阶段 ②：指挥基础训练

**目标**：学会怎么选中队友 → 下命令 → 队友执行 → 集合回来。

**场景**：训练大厅左中区域。

```
┌─────────────────────────────────────────┐
│  指挥基础区                              │
│                                         │
│          B ●                             │
│          (队友B)                         │
│                                         │
│    A ●                      ← 警戒点    │
│   (队友A)                               │
│                                         │
│       ①       ②        ③               │
│       ▲       ▲        ▲                │
│    地面标记点A  B       C               │
│                                         │
│         ← 玩家在起点位置                   │
│                                         │
│         ← 黄色「集合」圈                  │
│                                         │
└─────────────────────────────────────────┘
```

**教学顺序（逐步解锁）**：

| 步骤 | 教学内容 | 手势 | 玩家操作 |
|------|---------|------|---------|
| 2-1 | 选中队友 A | ☝️ 伸 1 指 | 右手靠近面部 ≤ 30cm → 伸 1 指 → 队友 A 高亮 |
| 2-2 | 选中队友 B | ✌️ 伸 2 指 | 伸 2 指 → 切换到队友 B 高亮 |
| 2-3 | 命令移动到标记 A | 👇 指向标记 A | 选中队友 → 指向地面标记 A → 队友走过去 |
| 2-4 | 命令加强警戒 | ✋ 手掌竖起 | 队友到达后 → 竖掌 → 队友盯住指定方向 |
| 2-5 | 命令集合保卫 | 👊 握拳拉回 | 握拳拉回胸口 → 队友走回玩家两侧保护 |
| 2-6 | 完整循环：A 去 A 点，换 B 去 B 点 | 组合 | 伸 1 指选 A→指挥 A→伸 2 指选 B→指挥 B |

**要点**：
- 所有手势必须把手靠近面部 ≤ 30cm（触发区内）才生效
- 选中状态下队友有轮廓高亮
- 玩家做对了 → 队友执行 → 提示"✅ 成功"
- 做错了 → 提示"请试试 XX 手势"并显示当前手势图示
- 全部 6 步完成才能解锁阶段③

---

### 阶段 ③：破门突入训练

**目标**：完成一次完整的 CQB 流程——选人 → 移到门 → 破门 → 清房。

**场景**：训练大厅右中区域。

```
┌─────────────────────────────────────────┐
│                                         │
│      ┌────── 小房间 ──────┐             │
│      │                    │             │
│      │      🎯 人形靶      │             │
│      │      (目标)         │             │
│      │                    │             │
│      └────────┬───────────┘             │
│               🚪                        │
│          训练门(已锁)                     │
│                                         │
│      ← 队友A                             │
│      ← 队友B                             │
│      ← 玩家                               │
│                                         │
└─────────────────────────────────────────┘
```

**流程**：

| 步骤 | 操作 | 手势 |
|------|------|------|
| ① | 选中队友 A（练习：一进门先选人） | ☝️ |
| ② | 指门 → 命令队友走到门边 | 👇 指向门 |
| ③ | 队友到位后 → 握拳（破门命令，面向门时自动识别为「踢门」） | ✊ 朝向门 |
| ④ | 队友踢开门 → 进入房间 → 扫描 → 喊"安全" | 自动执行 |
| ⑤ | 玩家进入房间 → 回到阶段②的指挥练习 | — |
| ⑥ | 练习同时指挥 A 和 B 配合破门 | 组合 |

**要点**：
- 门上有灯：红色 = 已锁，绿色 = 已破开
- 人形靶不会还击（只是站着）
- AI 会自动射击靶子（模拟真实战斗）
- 清完房间 → 屏幕显示"✅ 房间清理完成"
- 完成才能解锁阶段④

---

### 阶段 ④：综合考核

**目标**：在模拟实战中综合运用所有手势。

**场景**：训练大厅远端区域——一条短走廊 + 两个房间。

```
┌────── 房间 1 ──────┐   ┌────── 房间 2 ──────┐
│                     │   │                     │
│  👤 敌人（站桩）     │   │  👤 敌人（站桩）     │
│                     │   │                     │
│                     │   │                     │
└────────┬────────────┘   └────────┬────────────┘
         🚪                        🚪
         └──────── 走廊 ───────────┘
                                      ← 起点
```

**流程**：

```
计时开始
    ↓
玩家自行决定：谁走在前面？谁跟着？
    ↓
① 选人 → 指门 → 破门 → 清房 1 → AI 自动射击敌人
    ↓
② 选人 → 指门 → 破门 → 清房 2 → AI 自动射击敌人
    ↓
两个房间都清理完毕 → 计时停止
    ↓
显示成绩（评分标准见 MissionEvaluationSystem.md）
```

**敌人配置**：

| 房间 | 敌人数量 | 敌人类型 | 行为 |
|------|---------|---------|------|
| 房间 1 | 1 个 | 站桩靶子（不还击） | 站着不动 |
| 房间 2 | 2 个 | 站桩靶子（不还击） | 站着不动 |

**评分标准**（简化版）：

| 指标 | 要求 |
|------|------|
| ✅ 清理所有房间 | 两个房间都清完 |
| ✅ 无友伤 | 没打到队友 |
| ✅ 速度 | 30 秒以内完成 |
| ✅ 指挥次数 | 至少使用 3 种不同手势 |

**通过条件**：完成所有房间清理 + 无友伤 = 及格

**奖励**：通过后解锁「正式突击」模式（Assault Mode）。

## 四、关卡跳转逻辑

```
游戏启动
    │
    ├── 首次启动或手势模板为空
    │       → HandPoseRecordingLevel（先录手势模板）
    │       → TutorialLevel（进入教程）
    │
    ├── 已录手势但未完成教程
    │       → TutorialLevel（继续教程）
    │
    ├── 教程已完成
    │       → Home（主页）
    │            ├── 训练模式（重新挑战教程阶段④）
    │            ├── 突击模式（正式关卡，已解锁）
    │            └── 设置（含重录手势选项）
```

在 `FBlueHandCommandLevelRequest` 中：

```cpp
// 跳转到教程关卡
FTinyTroubleLevelRequest Request;
Request.TargetLevelName = "TutorialLevel";
Request.TargetPhase = EBlueHandCommandGamePhase::Tutorial;

// MetaData 记录从哪个阶段开始
Request.MetaData.Add("StartPhase", "1"); // 或 "2"/"3"/"4"
```

## 五、SaveGame 中的教程进度

```cpp
USTRUCT(BlueprintType)
struct FTutorialProgress
{
    GENERATED_BODY()

    // 阶段①：手势识别进度（7 个手势的分项记录）
    bool bGestureSelectA_Done = false;      // ☝️ 选中队友A
    bool bGestureSelectB_Done = false;      // ✌️ 选中队友B
    bool bGestureMoveTo_Done = false;       // 👇 移动到位置
    bool bGestureBreach_Done = false;       // ✊ 破门
    bool bGestureWatch_Done = false;        // ✋ 加强警戒
    bool bGestureHoldFire_Done = false;     // 🤚 禁止开火
    bool bGestureReturn_Done = false;       // 👊 集合保卫

    // 阶段②：指挥基础进度
    bool bCommandSelectA_Done = false;      // 伸1指选中A
    bool bCommandSelectB_Done = false;      // 伸2指选中B
    bool bCommandMoveTo_Done = false;       // 移动到位置
    bool bCommandWatch_Done = false;        // 加强警戒
    bool bCommandReturn_Done = false;       // 集合保卫
    bool bCommandCombined_Done = false;     // 组合指挥两个队友

    // 阶段③：破门训练进度
    bool bBreachTraining_Done = false;

    // 阶段④：综合考核
    bool bFinalExam_Done = false;
    float BestExamTime = 0.0f;

    // 整体完成
    bool bTutorialCompleted = false;
};
```

教程完成标记 `bTutorialCompleted` 控制 Home 关卡中「突击模式」的解锁。
