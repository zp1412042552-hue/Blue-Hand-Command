# ⚠️ 已废弃 — 手部姿势录制关卡

> **废弃原因**：新方案（[TacticalGestureCommandSystem.md](TacticalGestureCommandSystem.md)）采用 **5 个通用手势**，均为天然肢体动作（单指指天 / V字 / 指向 / 立掌 / 握拳拉回），无需录制个人模板即可稳定识别。
>
> 录制关卡在新方案中不再需要。本文档保留作为历史参考。

---

## 旧内容（仅供历史参考）

### 旧设计理念

旧方案（7 手势，基于 OculusHandPoseRecognition）需要玩家预先录制个人手势模板，因为手型姿态的细微差异会影响识别精度。

新方案将手势精简到 **5 个极度差异化的大动作**，每个手势之间的形态差距足够大，特征点角度匹配算法可以直接识别，不再需要个人校准。

### 旧方案 vs 新方案对比

| 维度 | 旧方案 | 新方案 |
|------|--------|--------|
| 手势数量 | 7 个 | **5 个** |
| 录制需求 | 必须预先录制模板 | ❌ 无需录制，开箱即用 |
| 识别方式 | OculusHandPoseRecognition 模板匹配 | 特征点角度匹配 + 时间滤波 |
| 误触防护 | 空间触发区（手必须靠近面部 ≤ 30cm） | 状态机被动激活 + 0.5 秒时间过滤 |
| 上下文感知 | 无 | ✅ 👇 根据 Raycast Tag 自动切换命令 |
