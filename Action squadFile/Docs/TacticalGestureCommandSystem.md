# 战术手势指挥系统 — GDD 核心设计方案

## 1. 设计概述

本系统是一套专为 VR/MR 射击游戏打造的 **"无安全区限制、上下文相关"** 的极简手势指挥系统。通过极限精简的 **5 个手势**，配合环境射线（Raycast）智能检测，让玩家在不依赖手柄的情况下，实现对 AI 队友的深度战术协同。

### 核心设计原则

**形态物理隔离**：不设置死板的"眼前触控区"。仅依靠手势本身的独特性与时间过滤（0.5 秒）防止非指挥动作（如持枪、换弹）的误触。

**上下文相关（Context-Sensitive）**：让环境变聪明。同一个"指"的手势，根据射线击中的物体标签（Tag）自动切换不同命令，极限压减玩家的学习成本。

**状态机被动激活**：命令手势平时处于锁死状态，只有成功锁定队友后才会激活检测，从底层逻辑上杜绝误触。

---

## 2. 核心 5 大手势列表

### 轴线一：全局选人手势（静默检测）

以下手势在任何舒适的空间位置均可触发，系统在后台保持静默检测。

| # | 手势 | 手势名称 | 动作说明 | 触发判定 | 战术功能 |
|---|------|---------|---------|---------|---------|
| ① | ☝️ | **单指指天** | 右手食指笔直指天，其余四指握拳 | 保持姿态 ≥ 0.5 秒 | **锁定队友 A**。成功后耳机播报"哔"提示音，HUD 队友 A 状态亮起 |
| ② | ✌️ | **V 字指天** | 右手食指与中指分开约 30° 指天，其余握拳 | 保持姿态 ≥ 0.5 秒 | **锁定队友 B**。成功后耳机播报"哔"提示音，HUD 队友 B 状态亮起 |

> **💡 多选/全选逻辑**：无需第三个选人手势。玩家锁定 A 后手不放下，紧接着做出 ②，HUD 会同时亮起 A 与 B，判定为"全选小队"。

---

### 轴线二：全局下令手势（被动激活）

> ⚠️ 手势 ③ 和 ④ 只有在**【轴线一】成功锁定队友后**才会激活检测。指令一旦发出，检测立刻关闭并清空选择。

| # | 手势 | 手势名称 | 动作说明 | 环境上下文判定（Raycast Tag） | 最终执行战术命令 |
|---|------|---------|---------|----------------------------|----------------|
| ③ | 👇 | **万能行动命令** | 右手食指指向战场目标，其余四指握拳，从手指/视线发射全息战术射线 | `Hit: "Floor"`（地面）<br>`Hit: "Door"`（木门/铁门）<br>`Hit: "Enemy"`（敌方单位） | ① 战术移动到该坐标并寻找掩体<br>② 前往门前执行突击破门/C4 准备<br>③ 全队集火该目标，将其列为首要消灭对象 |
| ④ | ✋ | **万能战术状态** | 右手五指自然张开，掌心朝前，手腕立起（经典 Stop 姿势） | 无需射线判定（原地生效） | **加强警戒 / 自由交火**：队友立刻就地寻找最近掩体，进入高阶戒备，允许自主索敌并开火 |

| # | 手势 | 手势名称 | 动作说明 | 核心特权 | 最终执行战术命令 |
|---|------|---------|---------|---------|----------------|
| ⑤ | 👊 | **全局召回命令** | 动态手势：右手握紧拳头，快速弯回胸口（类似拉重绳动作） | **无需提前选人**，任何时候做出均全局生效 | **全队集合保卫**：所有队友立刻放弃当前正在执行的一切任务，以战术 V 字阵型向玩家靠拢并贴身护卫 |

## 3. 底层交互状态机（State Machine）

为确保代码高效且绝无 Bug，后台逻辑必须严格遵循以下三种状态的单向/循环切换：

```
[状态 1：日常战斗]
       │
       ▼ (识别到 ☝️ 或 ✌️ 并保持 0.5 秒)
[状态 2：选择目标激活] ──► 耳机"哔" ──► 开启 👇 和 ✋ 的检测开关
       │
       ├─► (做出 👇 或 ✋) ───────► [执行命令] ──► 自动退回 [状态 1]
       ├─► (检测到 动态👊) ──────► [全队集结] ──► 自动退回 [状态 1]
       └─► (超时超过 2.0 秒无动作) ─► [选择清空] ──► 自动退回 [状态 1]
```

### 状态 1：日常战斗（静默状态）

**后台行为**：仅开启手势 ①(☝️)、②(✌️) 以及 动态⑤(👊) 的特征捕捉。

**游戏体验**：玩家正常开枪射击、拉栓、换弹。此时手部哪怕偶然做出 👇 或 ✋，系统也完全不予理会，保证核心射击体验的绝对流畅。

### 状态 2：选择目标激活

**后台行为**：当玩家做出 ☝️ 或 ✌️ 满足时间阈值，系统进入此状态。立刻关闭 ☝️ 和 ✌️ 的检测，同时向场景开启一条环境射线（Raycast Check），激活 👇 和 ✋ 的特征识别。

**视觉反馈**：视野上方淡入全息战术 HUD，高亮显示被锁定的队友头像。

### 状态 3：执行命令

**后台行为**：根据玩家做出的命令（👇 或 ✋），执行相应的 AI 行为树（Behavior Tree）脚本。

**退出机制**：
- **成功退出**：命令触发的瞬间，后台立即关闭射线与命令手势检测，数据清空，强制退回【状态 1】
- **防呆超时退出**：若玩家选人后由于战局变化放弃下令，只要手放下或保持静止超过 **2.0 秒**，系统自动清空当前选中的队友，淡出 HUD，退回【状态 1】

## 4. 与旧方案的关键区别

| 维度 | 旧方案（7 手势） | 新方案（5 手势） |
|------|----------------|----------------|
| 空间限制 | 手必须靠近面部 ≤ 30cm 触发区 | ❌ 无任何空间限制，任意位置均可 |
| 手势数量 | 7 个（选人×2 + 命令×5） | **5 个**（选人×2 + 命令×2 + 全局召回×1） |
| 破门操作 | 独立手势 ✊ | 合并到 👇 万能行动命令（Raycast 检测门） |
| 禁火/警戒 | 两个独立手势 ✋ 和 🤚 | 合并为一个 ✋ 万能战术状态 |
| 集合操作 | 需先选人再做 👊 | ⑤👊 全局召回，无需选人，任何时候生效 |
| 上下文感知 | 无（每个手势固定功能） | ✅ 👇 根据 Raycast Tag 自动切换 3 种命令 |
| 多选/全选 | 不支持 | ✅ ①② 连做 = 全选小队 |
| 误触防护 | 空间触发区隔离 | 状态机被动激活 + 0.5 秒时间过滤 |
| 选人超时 | 手离开触发区即取消 | **2.0 秒防呆超时**，手放下后仍可保持选择 |

## 5. Blackboard 数据结构

```cpp
// Blackboard 条目（每个队友独立）
├── SelfActor               本身引用
├── PlayerActor             玩家引用
├── CommandType             (None / MoveTo / Breach / FocusFire / WatchAndEngage / Recall)
├── TargetLocation          目标位置（指地面/门前/敌人位置）
├── TargetDoor              目标门（破门时）
├── TargetEnemy             目标敌人（集火时）
├── EnemyActor              当前敌人
├── bSelected               是否被玩家选中
└── bAllSelected            是否全选模式
```

手势组件只负责：**识别姿势 → 修改 Blackboard → Behavior Tree 自动响应**

## 6. 关键技术决策

| 决策 | 方案 | 理由 |
|------|------|------|
| 手势识别 | 特征点角度匹配 + 时间滤波 | 无需 Oculus 专有插件，通用 OpenXR 手部追踪 |
| 环境检测 | Line Trace（射线检测）by Channel/Tag | UE 内置，性能开销极低 |
| 寻路 | NavMesh（导航网格） | UE 内置，自动绕障，适用室内环境 |
| AI 决策 | Behavior Tree | UE 可视化节点搭建，无需手写复杂逻辑 |
| 数据通信 | Blackboard | UE 内置，与 Behavior Tree 天然配合 |
| 特殊动作 | Montage 动画（踢门/C4） | 播放动画序列，不需要 IK 或物理 |

## 7. 视觉与听觉反馈

| 事件 | 反馈 |
|------|------|
| 选中队友 A | 蓝色轮廓光 + 耳机"哔"一声（短促高音） |
| 选中队友 B | 黄色轮廓光 + 耳机"哔"一声（短促低音） |
| 全选模式 | A 蓝色 + B 黄色 同时亮起 + 双音效 |
| 👇 射线命中地面 | 地面出现蓝色全息移动标记圆环 |
| 👇 射线命中门 | 门上出现红色破门标记 + 锁图标 |
| 👇 射线命中敌人 | 敌人头上出现红色骷髅标记 |
| 命令执行成功 | 队友语音确认 + 指令线从手部连到目标 |
| 选择超时退出 | HUD 淡出 + 轻微衰减音效（提示选择已取消） |
| 👊 全局召回 | 所有队友身上闪烁绿色脉冲光 + 加速走回 |
